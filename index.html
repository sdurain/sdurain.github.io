Selected thesis projects

Uncertainty sensitivity snalysis with deep ensembles

Uncertainty Surrogate model Sensitivity
  • Goal: Quantify the feature importance for braking stability using a small simulation dataset.
  • Data generation: Low dataset physic simulation of braking stability (CEA simulation).
  • Approach: Surrogate model with a deep ensemble (10× MLP).
  • Result: Sensitivity analysis that accounts epistemic uncertainty.
  • Demo: Beam notebook ↗

RNN Surrogate Modeling for Nonlinear Braking Dynamics

Surrogate model Nonlinear dynamics
  • Data generation: Dataset generated with a high-fidelity ROM (Craig–Bampton).
  • Approach: RNN-based surrogate model trained on spectrogram
  • Result: High-fidelity predictions and demonstrates extrapolation capability.
  • Demo: Duffing oscillator notebook ↗

Brake Pad Semantic Segmentation

Semantic segmentation Transfer learning Transformer
  • Data generation: Multimodal dataset (optical and SEM) of brake-pad material with pseudo-annotations.
  • Approach: Fine-tune a SegFormer model via transfer learning.
  • Result: High-fidelity semantic masks enabling quantitative monitoring of brake-pad material.

Experimental & Numerical Data Acquisition (Brake Squeal)

Multimodal acquisition Signal processing ROM Dynamic Simulation
  • Goal: Build an experimental + simulated dataset to study and predict brake squeal occurrence.
  • Experimental: Multimodal measurements combining thermal, surface topography, static, and dynamic responses.
  • Numerical: ROM workflow enabling fast parameter sweeps and scalable synthetic data generation.
  • Outcome: Dataset used to train robust predictive models and support interpretability analyses.
